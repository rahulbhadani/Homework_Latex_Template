\section{Theory}
\subsection{Constraint Optimization (10 points)}
Consider the function $f(x, y) = e^{xy}$ subject to constraint $x^3 + y^3 = 16$. Use Lagrange multipliers to find the coordinates $(x, y)$ of any points subject to the given constraint where the function $f$ could attain a maximum or minimum.
% Source: https://canvas.harvard.edu/courses/376/files/18675/download?verifier=UF0kjDQdJNUypcJAiG8nlyzwLmCyIk0GAz30T3oL

\answer

\orangebox
{
We wish to find the extreme values of the function $f$ subject to the constraint $g(x,y) = x^3 + y^3 - 16$

Using Lagrange multiplier:

\begin{equation}
\label{eq:eq1}
    \begin{aligned}
        \nabla f & = \lambda \nabla g \\
        \Rightarrow ye^{xy} & = 3\lambda x^2\\
        \Rightarrow xe^{xy} & = 3\lambda y^2\\
      \end{aligned}
\end{equation}

In this case, if with either $x=0$ or $y=0$, it would make $x=0, y = 0$ based on Equation~\eqref{eq:eq1} and would violate our constraint $g(x,y) = x^3 + y^3 - 16$. Hence, clearly $x \neq 0$, $y \neq 0$.

Then

$$
\lambda = \cfrac{ye^{xy}}{3x^2} = \cfrac{xe^{xy}}{3y^2}
$$
based on Equation~\eqref{eq:eq1}.

That gives as $x^3 = y^3 \Rightarrow x = y$. Putting back this in our constraint, we get $2x^3 = 16 \Rightarrow x = 2, \Rightarrow y  = 2$.

Thus, at  the point $(2, 2)$, a point can attain optimum value.
}


\subsection{Relation between Constraint Optimization problem and SVM (5 points)}
Write down the formulation (i.e. mathematical equation) of the constraint optimization problem used in Support Vector Machine (SVM).


\answer

Constraint optimization problem in SVM is

$$
\text{minimize}~~~\cfrac{1}{2}||w||^2
$$

subject to 

$$
y_i (\wbf^\top \xbf_i + b) \geq 1
$$

$\forall  i = [1, n]$.

where the Data is $\{\xbf_i, y_i\}_{i = [1, n]}$.
